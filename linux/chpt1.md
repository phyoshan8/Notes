# Linux philosophy and concepts

  -kernel: glue between hardware and applications
  -(linux kernel);
  -distribution: distro
    -collection of software making up linux-based os
    -eg: debian, fedora, suse
  -boot loader: program that boots os
  -service: run as a bg process
    eg: httpd, nfsd, ntpd,
  -filesystem: method storing, organizing files
    eg:ext3, FAT, XFS, NTFS, Btrfs
  -X window system: tool for gui
  -DT env: GUI on OS
    eg: GNOME, KDE, XFce, Fluxbox;
  -command line: interface for tp command on OS
  -shell: cmd line interpreter, input cmd --> perform tsk
    eg: bash ,tcsh, zsch
  

  **linux: multitasking os**
  **distro include kernals and tools**
