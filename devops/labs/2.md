In the **Stratos Datacenter**, temporary developers (e.g., `jim`, `mariyam`, or `james`) often need server access for a specific project duration. To avoid "forgotten accounts" that create security holes, we set an **automatic expiration**.

- **Objective:** Automate account deactivation using an expiry date.
- **Key Concept:** [[Account Expiry]] (Self-disabling accounts).
